Project Descriptions

DnaSplicing : 
Creating a linked list from string input, using a variety of string operations, then traversing and printing the list

Flight: 
Cutting up string input, creating objects from the input and storing them in an ArrayList, traversing and printing the list

RockCountDown:
Simpler version of Flight

SortComparison:
Implementing Insertion Sort, Merge Sort, and QuickSort 

SpellChecker:
Building a binary tree from a text file, then traversing it to find a specific node, and outputting the results

XmlValidator:
Validating string input, then pushing that valid input to a self-implemented stack. Stack used in order to validate correct ordering of text xml tags or to show incorrect ordering
